## API Reference

#### Get all items

```http
  GET /api/https://api.themoviedb.org/3/search/movie?<your api key>
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `api_key` | `string` | **Required**. Your API key |
